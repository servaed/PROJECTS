# CLAUDE_CODE

Cloudera AI presales and SE workspace — projects built with Claude Code.

## Structure

| Folder | Purpose |
|---|---|
| `demos/` | Polished, deployable apps shown live to customers |
| `rfp/` | Tools and templates for RFP / RFI responses |
| `poc/` | Customer-specific proof-of-concept projects |
| `accelerators/` | Reusable code dropped into customer environments |
| `workshops/` | Hands-on lab materials for training sessions |
| `internal-tools/` | Productivity tools for the SE/presales team |
| `sandbox/` | Experiments and throw-away spikes |

## Naming convention

```
{customer-or-industry}-{technology}-{use-case}

demos/cloudera-ai-id-rag-demo        # Indonesia RAG + SQL demo
poc/bank-xyz-credit-scoring          # customer-specific PoC
accelerators/cdp-hive-connector      # reusable library
```

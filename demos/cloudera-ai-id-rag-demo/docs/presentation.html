↓ top-K chunks → LLM

Technical Deep Dive
AI Intelligence Modes

Seven Modes for Every Use Case

Each mode is a toggle in the input bar — composable, independent, with dedicated SSE event streams and frontend components.

Think — Reasoning Mode
Streams chain-of-thought via thinking_token SSE events. Works with DeepSeek-R1, Nemotron <think> tags and Anthropic extended thinking. ThinkingPanel shows live CoT.
Agent — Multi-Step Research
LLM Planner → parallel Executor (retrieve + SQL per step) → Synthesizer. Events: agent_plan / agent_step_* / agent_synthesis. Visible step trace in UI.
Debate — Adversarial Review
Researcher LLM gathers evidence → Critic LLM challenges (streaming) → Synthesis resolves. Events: debate_researcher_done / debate_critic_token.
Vision — Multi-modal Input
Base64 image attached in ChatRequest.image_b64, injected as OpenAI vision content block in synthesis prompt. Works with GPT-4o, Claude 4, Gemini.
Map — Geographic Heatmap
SQL result with city/region/province column auto-triggers Leaflet bubble map. 27-city + 24-province coordinate lookup. CartoDB tiles, color-coded by metric value. Offline-safe (self-hosted Leaflet).
Forecast — Predictive Analytics
POST /api/forecast runs OLS linear regression on time-series SQL results. Projects 3 periods ahead with dashed overlay on the line chart. Returns trend (↑↓→) and R² score.

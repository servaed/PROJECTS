# Changelog — 2026-05-04

## Features Added
- **Indonesia Map Heatmap** — Leaflet 1.9.4 (local), MapChart component, auto-detection of geo columns, 27-city + 24-province coordinate lookup, bubble markers with color encoding, CartoDB tiles
- **Reasoning Mode (Think)** — Think toggle, ThinkingPanel showing chain-of-thought, `thinking_token` SSE events, works with `<think>` tag models and Anthropic extended thinking
- **AI Debate Mode** — `POST /api/debate/chat`, Researcher+Critic+Synthesis pipeline, DebatePanel component with two-card layout
- **Document Intelligence** — pdfplumber PDF table extraction, `src/retrieval/table_extractor.py`, auto-extraction on PDF upload, tables registered as DuckDB views with `doc_` prefix

## Data Enrichment
- 2 new tables: `loan_application` (600 rows), `network_incident` (162 rows)
- New columns: customer (industry, annual_revenue, debt_service_ratio), branch (npl_amount, deposit_balance, roi_pct), subscriber (tenure_months, monthly_complaints), network (avg_latency_ms, packet_loss_pct), public_service (pending_count, complaint_count)
- NPL tiers: Java metro 3-6%, Sumatra/Kalimantan 7-12%, outer islands 13-22%
- 22 complex sample questions + 2 cross-domain questions
- 15+ new SQL few-shot examples including JOINs, HAVING, conditional aggregation, derived metrics

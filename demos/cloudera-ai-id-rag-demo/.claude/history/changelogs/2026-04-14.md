## Session 1 — Feature completion and quality improvements

### UI (app/static/index.html)
- **Feature #5: Source document preview with keyword highlighting**
  - `highlightHtml(text, query)` — HTML-escapes raw chunk text then wraps matched query words (≥3 chars) in `<mark>` tags; safe against XSS
  - `DocCard` upgraded with **▼ Show full chunk / ▲ Show excerpt** toggle; shows `full_text` from SSE payload
  - `ASSISTANT` reducer case stores `question` field; `handleSubmit` dispatches `question`; `DocCard` receives `query` prop
- **Feature #4: Demo auto-play mode**
  - Three refs (`autoPlayRef`, `samplesRef`, `handleSubmitRef`) avoid stale-closure bugs in `setTimeout` callbacks
  - `startAutoPlay()` / `stopAutoPlay()` in `App`; `handleSubmit` accepts `_onDone` callback
  - **▶ Run Demo** / **⏹ Stop Demo** button with pulsing indicator and question counter in `Sidebar`
- **New Conversation button** — trash icon in topbar; dispatches `RESET` + clears `sessionStorage`
- **Copy answer button** — appears on every completed assistant message; `IcoCopy` SVG icon
- **Session storage persistence** — chat survives F5; streaming messages clamped to `isStreaming:false` on reload
- **Block input during auto-play** — `InputBar` and sidebar picks disabled while `isAutoPlaying`
- **`⚙ Configure` link** added to topbar (links to `/configure`)
- **`⚙ Configure` button** added to `setup.html` topbar

### Backend (app/api.py)
- Fixed `/api/status` LLM indicator: Bedrock/Anthropic providers with no `LLM_BASE_URL` now correctly return `ok: true`

### Tests
- `tests/test_api.py` — 12 new tests covering FastAPI endpoints, SSE stream shape, event ordering, and LLM provider indicator bug

### Deployment
- `Dockerfile` + `.dockerignore` added for container-based deployment path

## Session 2 — Configure wizard

### New: `/configure` environment variable wizard
- `GET /configure` — serves `app/static/configure.html`
- `GET /api/configure` — returns current config state; secrets masked as `●●●●●●●●`; each key has `value` + `source` (`env` / `file` / `null`)
- `POST /api/configure` — validates keys against `_CONFIGURE_ALLOWED_KEYS`; skips platform env vars with warning; writes `data/.env.local`; updates `os.environ` immediately
- `_OVERRIDE_PATH = Path("data/.env.local")` in `api.py`

### New: `app/static/configure.html`
- Provider selector (Cloudera / OpenAI / Bedrock / Anthropic / Local) — dynamically shows/hides fields
- Source badges: 🟢 From environment (locked) / 🔵 From saved file / ⬜ Not set
- Password reveal toggles for API key, DB URL
- Bedrock note explaining AWS env vars are managed separately
- Save → success banner with restart reminder; badge reload after save

### Updated: `deployment/launch_app.sh`
- New step [0/5]: sources `data/.env.local` before uvicorn
- Platform env vars take precedence (skip + warn if already exported)

### Documentation refresh

- `DEPLOYMENT.md` — full rewrite with configure wizard flow, updated startup log examples, new troubleshooting rows, consolidated env var reference

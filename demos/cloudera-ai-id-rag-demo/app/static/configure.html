Environment Configuration

Set your LLM credentials and infrastructure settings here. Values are saved to data/.env.local and applied immediately. Restart the application for full effect (new DB connections, embedding model reload).
Environment variables set via the Cloudera AI platform UI always take precedence and are shown as From environment.
